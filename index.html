当前支持规则

目标是保留正文结构，删除主题和扩展相关表现层语法。

Admonitions / Details转为 blockquote，标题加粗。
Content tabs转为可移植的小标题和正文。
Buttons / Attribute lists保留链接、图片和文本，移除样式属性。
Annotations正文标记转为 [1]，代码行末 annotation 注释被移除。
Icons / Emojis默认移除 Material 系图标 shortcode，可选移除全部 shortcode。
Grids / Cards / Captions展开包装，保留内部 Markdown 和说明文字。
Formatting / Keys高亮、critic markup、键盘组合等降级为常见 Markdown。
YAML front matter标题、摘要、日期、标签等转换为标准 Markdown。
More marker移除博客摘要分隔符 <!--more-->。
Snippets浏览器无法读取本地文件，转为注释占位。
